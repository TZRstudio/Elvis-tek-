# Etiquetas del proyecto

## Formulario
La usamos para issues relacionados con la creación o modificación de formularios HTML.

## Requerimientos
La usamos para issues que describen funcionalidades pedidas por el cliente.

## Validación
La usamos para issues que necesiten control y verificación de datos ingresados.

## Experiencia de usuario
La usamos para issues que mejoran la interacción y usabilidad del sitio.

## Input
La usamos para issues relacionados con campos de entrada de datos.

## Archivo
La usamos para issues que involucran carga o manejo de archivos (PDF).

## Login
La usamos para issues que esten relacionados con el acceso al sistema interno de empleados.
