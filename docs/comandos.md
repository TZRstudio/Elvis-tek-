# Comandos Git que usamor en el proyecto

## Configuración inicial
git init                        # Inicializa el repositorio
git remote add origin https://github.com/TZRstudio/Elvis-tek-     # Conecta con el repositorio remoto

## Trabajo diario
git status                      # Ver el estado de los archivos
git add .                       # Agregar todos los cambios
git add [archivo]               # Agregar un archivo específico
git commit -m "mensaje"         # Guardar los cambios con descripción
git push origin main            # Subir los cambios a GitHub
git pull origin main            # Bajar los cambios de GitHub

## Revisión
git log                         # Ver historial de commits
git log --oneline               # Ver historial resumido
